Minigame2(Maze)

Info

Lógica del juego

Este minijuego se basa en una matriz 2D que representa un océano donde un pez globo (P) debe moverse hasta alcanzar la comida (C), el pez debe evitar los obstáculos (X). El jugador controla el pez ingresando las teclas W, A, S, D para moverse en diferentes direcciones.

Como funciona

1. Inicialización:

o Se define un océano como una matriz de 5x5.

o El pez (P) comienza en una posición. (matriz)

o Los obstáculos (X) están colocados estratégicamente.

o La comida (C) está en un lugar del entorno (matriz).

2. Ciclo de Juego

o Se muestra el estado actual del océano en la consola.

o Se verifica si el pez ha llegado a la comida (C). Si es así, el juego termina con un mensaje de ganador

o Se pide al usuario una dirección de movimiento (W, A, S, D).

o Se intenta mover el pez. Si el movimiento es válido (sin chocar con los obstáculos (X) o salirse de los límites), el pez se mueve.

o Si el movimiento es inválido, se muestra un mensaje de advertencia.

o El objetivo es alcanzar C evitando X.

3. Movimiento del Pez globo

o Se calcula la nueva posición según la entrada del usuario.

o Se verifica si la nueva posición está dentro del área y no hay un obstáculo (X).

o Si el movimiento es válido, el pez deja su posición anterior ('-') y se mueve.

o Si el jugador presiona W → El pez se mueve hacia arriba

o Si el jugador presiona A → El pez se mueve hacia la izquierda.

o Si el jugador presiona S → El pez se mueve hacia abajo.

o Si el jugador presiona D → El pez se mueve hacia la derecha.

o Si el jugador presiona (WASD) hacia una X → Movimiento inválido (obstáculo X).
